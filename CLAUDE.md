# Claude Configuration

## Project: DeFi10
DeFi portfolio tracker with multi-protocol support (AAVE, Kamino, Uniswap, Pendle, Raydium).

## Tech Stack
- Frontend: React, Vite, TypeScript, TailwindCSS
- Backend: Rust, Axum, SQLx, Tokio

## Critical Rules

1. **NO comments** unless asked
2. **NO .md files** unless asked
3. **English only** for all code
4. **Implement directly** - don't suggest

## Common Tasks

### Frontend
- Components in `frontend/src/components/`
- Services in `frontend/src/services/`
- State with React hooks

### Backend Rust
- Handlers in `backend-rust/src/handlers/`
- Services in `backend-rust/src/services/`

## Testing
- Run frontend tests: `npm test`
- Run rust tests: `cargo test`
